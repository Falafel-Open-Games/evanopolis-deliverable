# EVANOPOLIS Visual System Summary

## Overview
The EVANOPOLIS background pack consists of six regional illustrations representing sustainable Bitcoin mining across global energy archetypes. Each image follows a unified visual language designed for UI integration in a strategy game environment.

---

## Global Style Lock

### Core Principles
- **Flat vector illustration only** — no texture, grain, or painterly effects.
- **Clean geometric shapes** — simplified forms, minimal detail.
- **Soft gradients** — subtle tonal transitions, no harsh contrast.
- **Desaturated palette** — calm, modern, and optimistic color mood.
- **Consistent composition** — open center for UI overlay, balanced side elements.

### Composition Rules
| Element       | Guideline                                |
|---------------|------------------------------------------|
| Aspect Ratio  | 16:9 wide format                         |
| Center        | Open and uncluttered for UI placement    |
| Depth         | Clear layering: foreground / midground / background |
| Balance       | Elements distributed left and right, horizon line stable |
| Focus         | Calm, readable at small scale            |

## Color System

### Palette Characteristics
- Slightly desaturated hues for a professional tone.
- Soft gradient transitions (no hard edges).
- Avoid pure black or pure white; use tinted neutrals.

### Common Gradient Directions
- **Sky:** vertical gradient (light horizon → deeper upper tone)
- **Terrain:** horizontal or diagonal gradient for subtle depth
- **Water:** gentle horizontal gradient (light center → darker edges)

---

## Consistency Guidelines
- Maintain identical line weight and simplification level across all scenes.
- Use the same horizon height and perspective depth.
- Keep cloud shapes, vegetation, and structural geometry stylistically uniform.
- Ensure all compositions have an open central area for UI overlays.

---

## Technical Notes
- Vector format recommended: **SVG or AI** for scalability.
- Avoid raster effects, transparency blending, or texture overlays.
- Each file should be exported with identical dimensions and color profile.

---

## Visual Tone Summary
- **Mood:** calm, optimistic, modern.
- **Purpose:** serve as unobtrusive UI backgrounds.
- **Identity:** unified global aesthetic reflecting sustainable energy innovation.

---

## Future Expansion Guidelines
- New regions should follow the same geometric simplification and palette logic.
- Introduce new energy archetypes (e.g., tidal, biomass) using existing compositional rhythm.
- Keep the center open and maintain visual balance between natural and industrial elements.

---

**EVANOPOLIS Visual System** establishes a cohesive foundation for scalable, elegant UI backgrounds that communicate global sustainability through minimalist design.
